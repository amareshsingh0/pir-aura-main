# PIR Aura

PIR Aura is a motion detection dashboard built with React, Vite, and TailwindCSS. It provides real-time motion data, history tracking, and customizable alerts for motion detection systems.

## Features
- Real-time motion detection updates
- History tracking with export options (CSV)
- Sound and voice command settings
- Responsive and modern UI

## Technologies Used
- **React**: Frontend library for building user interfaces
- **Vite**: Build tool for fast development
- **TailwindCSS**: Utility-first CSS framework
- **TypeScript**: Static typing for JavaScript

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/amareshsingh0/pir-aura-main.git
   ```
2. Navigate to the project directory:
   ```bash
   cd pir-aura-main
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
1. Start the development server:
   ```bash
   npm run dev
   ```
2. Open the application in your browser at `http://localhost:8080`.
